# Agentic Web - 50 Project Ideas

基于官方 [Problem Statement](https://mystenlabs.notion.site/agentic-web-problem-statement)，每个项目必须展示 Sui 在 AI 系统中的具体价值（Move 对象、zkLogin、PTB、DeepBook、Walrus、Seal），通用 LLM 套壳不会入选。

---

## 风险监控与安全

1. **DeFi 风险哨兵** — 接入 Sui 借贷协议预言机，AI 模型实时计算风险评分，超阈值自动通过 Move 策略对象暂停市场，DAO 可覆盖
2. **闪电贷攻击检测器** — 监控链上 mempool 交易模式，AI 识别闪电贷攻击特征，通过 PTB 自动触发保护机制
3. **预言机异常监控** — AI 监控多个预言机价格源，检测偏差和操纵，异常时自动告警并建议协议参数调整
4. **智能合约漏洞扫描 Agent** — 分析 Move 合约字节码，AI 识别常见漏洞模式，生成安全报告和修复建议
5. **链上反洗钱 Agent** — 追踪可疑资金流向，AI 分析地址行为模式，通过对象所有权模型标记风险地址
6. **协议健康度仪表盘** — 聚合 TVL、流动性深度、清算数据，AI 生成协议健康评分和风险预警
7. **Gas 费优化 Agent** — 分析历史 Gas 模式，AI 推荐 PTB 打包策略和最佳提交时机

## Agent 钱包与自主执行

8. **zkLogin Agent 钱包** — 使用 zkLogin 为 AI Agent 创建受限钱包（预算上限、协议白名单、时间限制），链上活动全记录
9. **DCA 定投机器人** — 用户设定定投策略，Agent 自动通过 DeepBook 执行定期兑换，预算自限、可撤销
10. **DeFi 收益自动复投** — Agent 监控收益池 APY 变化，自动将收益复投到最优池，通过 PTB 原子化执行
11. **多策略交易 Agent** — Agent 根据市场信号在 DeepBook 上执行多种交易策略，每笔交易受 Move 策略对象约束
12. **跨境支付 Agent** — AI 解析支付意图，自动选择最优路径（DEX 路由、桥接），通过 PTB 原子化完成多步支付
13. **NFT 地板价狙击 Agent** — 监控目标集合地板价，达到阈值时 Agent 自动购买，预算上限保护
14. **流动性管理 Agent** — 自动调整 DEX LP 仓位范围，AI 根据波动率预测建议集中或分散流动性

## 意图引擎

15. **自然语言交易引擎** — 用户输入"用 100 SUI 换 USDC 并存入 Scallop"，AI 解析为 PTB 并执行，带风险守护检查
16. **财务目标规划器** — 用户描述储蓄目标，AI 编排定期买入 + 收益农耕的 PTB 策略
17. **多步 DeFi 编排器** — "借 USDC → 买 BTC → 质押 → 对冲" — AI 将复杂 DeFi 操作编译为单个 PTB
18. **投资组合再平衡器** — 用户设定目标配比，AI 定期生成再平衡 PTB，展示预期变化后由用户确认
19. **薪资自动分配器** — 收到工资后，AI 按用户描述的规则自动分配到储蓄、投资、支付等子钱包
20. **链上遗嘱/条件触发器** — 用户设定条件（如长期未活跃），AI 监控并自动触发资产转移 PTB

## 数据分析与预测

21. **链上情感分析 Agent** — 抓取社交数据和链上交易，AI 生成市场情感指数，关联 Sui 生态代币价格
22. ** whale 钱包追踪器** — AI 识别和标记大户地址，追踪其链上操作，生成实时告警和策略信号
23. **DeFi 收益排行榜** — 聚合 Sui 生态所有收益池，AI 预测未来 APY 趋势，推荐最优策略
24. **NFT 稀有度与估值 Agent** — 分析链上 NFT 属性和交易历史，AI 生成估值报告和稀有度排名
25. **链上数据问答 Bot** — 用户用自然语言提问（"过去 24 小时 Sui 上交易量最大的协议是哪个？"），AI 查询链上数据并回答
26. **协议用户行为分析** — AI 分析协议用户画像、留存率、活跃度，帮助项目方优化产品
27. **跨链套利机会扫描** — AI 监控 Sui 与其他链的价差，发现套利机会时生成 PTB 策略

## 开发者工具

28. **Move 合约 AI 审计助手** — 输入 Move 代码，AI 逐函数分析安全性，标注风险等级和改进建议
29. **PTB 可视化构建器** — 拖拽式界面构建 PTB，AI 辅助推荐下一步操作和风险提示
30. **Sui 开发 Copilot** — VS Code 插件，AI 辅助编写 Move 合约，自动补全 Sui 特有模式（对象、能力、PTB）
31. **智能测试生成器** — 分析 Move 合约接口，AI 自动生成测试用例，覆盖边界条件
32. **Gas 分析优化器** — 分析 PTB 执行路径，AI 推荐 Gas 优化方案，预估节省百分比
33. **合约文档自动生成** — 从 Move 代码自动生成中文/英文 API 文档和使用示例
34. **事件监控与告警框架** — 开发者自定义规则，AI Agent 监控链上事件并触发告警或自动响应

## 社交与社区

35. **链上社交信用评分** — 基于链上行为（交易历史、NFT 持有、DAO 参与）AI 生成去中心化信用分
36. **DAO 治理 AI 助手** — 分析提案内容，AI 生成摘要、影响分析和投票建议
37. **社区贡献追踪器** — AI 追踪和评估成员在 Sui 生态的贡献（代码、内容、社区活动），生成贡献度排名
38. **去中心化内容审核 Agent** — 社区提交的内容由 AI 初审，有争议的提交链上投票，结果写入 Move 对象

## 游戏与 NFT

39. **AI NPC 经济 Agent** — 游戏中的 AI NPC 在 Sui 上自主交易资源，影响游戏经济系统
40. **动态 NFT 进化 Agent** — AI 根据持有者行为和环境数据动态调整 NFT 属性和外观
41. **游戏资产自动交易 Agent** — 玩家设定买卖规则，Agent 在游戏资产市场上自动执行

## 基础设施

42. **多 Agent 协作框架** — 在 Sui 上构建多 Agent 协调系统，Agent 间通过 Move 对象传递任务和状态
43. **去中心化 AI 推理市场** — 将 AI 推理请求和结果上链，通过 Sui 对象模型管理模型版本和付费
44. **Agent 身份与声誉系统** — 基于 Sui 对象为 AI Agent 建立链上身份、能力证明和声誉评分
45. **可信 AI 输出预言机** — AI 推理结果通过零知识证明验证后上链，其他合约可安全引用
46. **模型训练数据市场** — 基于对象模型的数据集交易市场，AI 训练任务通过 PTB 管理数据授权和支付

## 综合应用

47. **个人财务 AI 助手** — 接入用户 Sui 钱包，AI 分析资产分布、收益情况，自动推荐和执行优化策略
48. **供应链金融 Agent** — AI 评估供应链风险，自动触发链上付款、开立信用证，通过 PTB 原子化结算
49. **保险理赔自动 Agent** — AI 评估理赔条件（链上数据 + 外部数据），满足条件自动通过 PTB 执行赔付
50. **去中心化预测聚合器** — AI 聚合多个数据源的预测结果，生成置信度评分，结果上链可审计
